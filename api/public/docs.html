Book Store API Documentation

Base URL: http://localhost:8000/api/v1

All responses use: { "success": true|false, "message": "...", "data": {...} }

Auth header: Authorization: Bearer <token>

Public Catalog

No authentication required

GET /books — List books (paginated)
GET /books/{id} — Get book by ID
GET /categories — List categories
GET /categories/{id} — Get category
GET /authors — List authors
GET /authors/{id} — Get author

Employee Auth

POST /employees/login
Body
emailstring, required
passwordstring, required

Requires: Authorization: Bearer <employee_token>

POST /employees/logout
POST /employees/refresh
GET /employees/me
GET /employees/orders — List orders (employee)
GET /employees/orders/{id}
PATCH /employees/orders/{id}/status

Customer Auth

POST /customers/register
Body
namestring, required
emailstring, required, unique
passwordstring, min 8, confirmed
addressstring, optional
countrystring, optional
citystring, optional
phonestring, optional
POST /customers/login
emailstring
passwordstring

Requires: Authorization: Bearer <customer_token>

POST /customers/logout
POST /customers/refresh
GET /customers/me
PUT /customers/profile — Update profile

Cart (Customer)

GET /customers/cart
POST /customers/cart/items
book_idstring, required
quantityinteger, optional, default 1
DELETE /customers/cart/items/{bookId}
PATCH /customers/cart/items/{bookId}
quantityinteger

Orders (Customer)

POST /customers/orders/checkout
shipping_addressobject: address, city, country, postal_code (optional)
payment_infoobject, optional
GET /customers/orders
GET /customers/orders/{id}
PATCH /customers/orders/{id}/status

Admin

Requires: Employee token + role: manager, shipping, review, accounting, employee

GET /admin/books
POST /admin/books
GET /admin/books/{id}
PUT /admin/books/{id}
DELETE /admin/books/{id}

Book payload: title, author_ids[], category_id, price, isbn, warehouse_id, stock_quantity, pages, publish_year, etc.
